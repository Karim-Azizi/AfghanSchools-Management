# Afghan Reconstruction Data

This repository contains the datasets, analysis scripts, and supplementary materials for the study:

**Title:** Managerial Diplomacy: The Role of Strategic Leadership in Reconstructing Afghanistan's Economic and Political Engagement with the Global Community

**Author:** Karim Azizi  
**Affiliation:** Department of Management, Faculty of Economics and Business, University of Lampung

## Contents

- `Wave1_Survey.csv` – Initial survey responses (25 stakeholders, 321 individual responses)  
- `Wave2_Interviews.csv` – Follow-up interviews (same 25 stakeholders, 121 responses)  
- `analysis_scripts/` – Python code for preprocessing, thematic analysis, sentiment analysis, clustering, and topic modeling  
- `figures/` – Publication-quality figures  
- `tables/` – Comprehensive analysis tables  
- `supplementary_material/` – Survey instruments and methods details
